<h1 className={"openapi__heading"}>Publishes a payment terminal configuration to the Handpoint Gateway.</h1>

<MethodEndpoint method={"post"} path={"/partner/partner/{partnerId}/publish/{terminalType}/{serialnumber}"}></MethodEndpoint>



Using data from the partner, acquirer and merchant this endpoint will push the terminal configuration to the Handpoint Gateway.  For the terminal to properly start up and initialise,  it is required for the configuration to be published to the Handpoint Gateway.
