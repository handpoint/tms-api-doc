<h1 className={"openapi__heading"}>Removes an existing payment terminal from its merchant.</h1>

<MethodEndpoint method={"delete"} path={"/partner/partner/{partnerId}/terminal/{terminalType}/{serialnumber}"}></MethodEndpoint>



This endpoint will unassign a terminal from its merchant. It is necessary to publish the configuration after this operation  so we can keep up to date the config published to the Handpoint Gateway. It will leave the terminal unassigned from any merchant and will become part of the pool of terminals ready to be assigned.
